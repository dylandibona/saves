## 4. Enrichment pipeline by platform

Architecture in `lib/enrichment/enrich.ts` (~700+ lines after this session). Hot loop:

1. `detectUrlType` from `lib/utils/url-detect.ts` returns one of: `google_maps | instagram | youtube | tiktok | spotify | apple_music | apple_podcasts | letterboxd | goodreads | generic`
2. `pickUserAgent(url)` chooses UA per-host
3. Routes to one of ten branches; each can use `fetchOEmbed` + `fetchAndParse` + `classifyWithClaude(url, og, html, hint?)`

| Platform | Detection | Quality | Notes |
|---|---|---|---|
| **Google Maps** | `maps.google.com`, `maps.app.goo.gl`, `goo.gl` | ✅ excellent | Places API (New) `searchText` with 500m bias; resolves photos with `skipHttpRedirect=true`. **Gating:** `GOOGLE_PLACES_API_KEY` needs Vercel env. |
| **Generic articles** | fall-through | ✅ excellent | Slackbot UA wins Cloudflare/Bot-protected sites. OG fallback in `fetchAndParse` tolerates non-2xx with >5KB body. Recipe JSON-LD passed to Claude as raw text. |
| **YouTube** | `youtube.com` / `youtu.be` / `m.` / `music.` | ✅ shipped this session | oEmbed → title + channel + maxres thumbnail. Claude with `'video'` hint picks correct category. |
| **TikTok** | `tiktok.com` / `vm.` / `vt.` | ✅ shipped this session | oEmbed + Slackbot UA fallback. `@handle` subtitle. |
| **Spotify** | `open.spotify.com` / `spotify.link` | ✅ shipped this session | Path-kind extraction → music vs podcast routing. |
| **Apple Music** | `music.apple.com` | ✅ shipped this session | Slackbot UA + `'music'` Claude hint |
| **Apple Podcasts** | `podcasts.apple.com` | ✅ shipped this session | Extracts `showName` / `episodeNumber` |
| **Letterboxd** | hostname | ✅ shipped this session | Slackbot UA + `'movie'` hint |
| **Goodreads** | hostname | ✅ shipped this session | Slackbot UA + `'book'` hint |
| **Instagram** | `instagram.com` / `instagr.am` | 🟡 mid | Facebook bot UA catches OG image + sometimes caption. Misses comments. Apify is the unblock (~$30/mo). |
| Twitter/X | n/a | ⬜ skipped | Low scrape success rate; would ship hollow |
| News paywalls (NYT/WSJ) | falls through to generic | 🟡 sometimes 403s | Apple News / IG-shared news links often land as link-only |

**New since 2026-05-17:**
- `decodeHtmlEntities()` in enrich.ts handles named entities + numeric entities + Latin diacritics. Fixes `&#39;` artifacts in older saves.
- 20s hard timeout on Anthropic classify call.
- `saves.enrichment_errors jsonb` populated by share-save when fallback signals fire — query with `SELECT * FROM saves WHERE enrichment_errors IS NOT NULL`.
- Persistent hero images via `lib/enrichment/image-persist.ts` → sharp resize → Supabase Storage `hero-images/{saveId}.webp` → public URL. Inline call from both `addSave` and `share-save`. Detail prefers Storage URL, falls back to source.

## 5. Account / billing model

### Roles

| Role (`household_members.role`) | Sees `/billing`? | Pays? | App experience |
|---|---|---|---|
| `owner` | Full UI | Yes | Library + InvitesSection + TestersSection |
| `member` | "Handled by {owner}." tile | No (owner covers seat) | Identical library — no billing surface |
| Solo | Same as `owner` | Yes | Same as today |

### Plans (`lib/billing/stripe.ts`)

| Plan | Price | Used by |
|---|---|---|
| `null` + `status='free'` | $0 (12-save cap) | Default new signups |
| `'personal'` | $4/mo or $36/yr | Owners on paid plans, or comp testers |
| `'household'` | **$8/mo flat for up to 4** | Owners hosting shared library |

Pricing changed this session from `household_member` $2/mo per seat → flat `household` $8/mo. `getPriceIds()` reads `STRIPE_PRICE_ID_HOUSEHOLD` with `_HOUSEHOLD_MEMBER` env fallback for legacy values.

### Trial mechanics

- App-code redemption grants 90-day comp Personal trial + stamps `users.acquired_via_code`.
- `/billing` countdown banner: ruby ≤7 days, amber ≤21, teal otherwise.
- `/settings` TestersSection lists redeemed app codes joined with redeemer expiry, sorted closest-first, color-coded urgency (`ok / soon / urgent / expired / inactive / unknown`).
- Conversion path: query cohort with `acquired_via_code IS NOT NULL`, mint fresh founder codes when Stripe goes live.

### Gate state

`BILLING_ENFORCED` env var = unset → `userCanSave` always returns `{ok:true}`. Code paths through `lib/billing/can-save.ts` exist and are wired into `addSave` + `/api/share-save`. One env flip activates enforcement.
